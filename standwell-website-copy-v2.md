# DESIGN & IMPLEMENTATION NOTES

## Visual Direction
- **Clean, minimal, confident.** Not flashy, not cheap. Think Stripe meets trade show.
- **Photography:** Product shots on white or in context (booth setups). Avoid stock photos of smiling salespeople.
- **Color:** Keep it simple — one primary brand color, black/white, minimal accents.
- **Typography:** Modern sans-serif (Inter, DM Sans, or similar). Professional but approachable.

## Trust Signals to Add
1. **Logo bar** — Even 3-4 company logos builds credibility. Ask first customers for permission.
2. **Testimonials with photos** — Face + name + company = more trust than anonymous quotes.
3. **"Trusted by X+ companies"** — Use real number once you have it. Start with "Trusted by growing teams" if needed.
4. **Payment badges** — Stripe, PayPal logos signal legitimate checkout.
5. **Satisfaction guarantee badge** — Even a simple "Quality Guaranteed" icon helps.

## Key Changes from Original
1. **Hero repositioned** — Now leads with pain point (no BS) instead of generic benefit
2. **Added "Who We Work With"** — Visitors can see themselves in a segment
3. **Added "How It Works"** — Simple 3-step process builds confidence
4. **Added testimonials section** — Social proof throughout the page
5. **Added guarantee section** — Reduces purchase anxiety
6. **Booth Kits added** — Higher-ticket productized offering
7. **Pricing adjusted** — Bumped minimums slightly to support quality positioning
8. **Footer enhanced** — Trust badges, better navigation
